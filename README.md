📊 Dashboard 1: Sales Overview
🔹 Description

The Sales Overview Dashboard provides a comprehensive snapshot of e-commerce fashion business performance. It highlights key metrics such as total sales, total orders, average order value (AOV), total quantity sold, and cancellation rate.

The dashboard enables quick analysis of monthly sales trends and product category performance, helping identify revenue patterns and high-performing segments. Interactive slicers for month, category, and delivery type allow users to dynamically explore the data and gain meaningful insights for decision-making.

🔹Key Highlights
KPI tracking (Sales, Orders, AOV, Quantity, Cancellation Rate)
Monthly sales trend analysis
Category-wise performance insights
Interactive filtering for dynamic exploration
